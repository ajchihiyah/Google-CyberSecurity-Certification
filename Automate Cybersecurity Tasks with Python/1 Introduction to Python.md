## Advantages

- Resembles human language
- Less code
- Easy to read
- Standard guidelines
- Large online community
- Built in standard libraries

## Python commands

- `type(x)`: Returns data type of a variable.
- `for i in range(10)`: Repeats action 10x.
- `while x <= 10`: Repeats until x > 10.
